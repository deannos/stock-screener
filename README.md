# 📈 Fundamental Stock Screener (India)

This project screens fundamentally strong Indian stocks using real-time financials, macroeconomic insights, and investor preferences.

## 🔍 Features

- Aligns with macroeconomic stages: Recession, Recovery, Expansion
- Filters by ROIC > CoE and FCF Yield > 3%
- Uses yfinance for real-time data
- Interactive UI using Streamlit
- Bubble chart: ROIC vs FCF Yield
- Download results as CSV

## 🚀 Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
